Programming Projects

Learning by building from scratch.

Programming gave me another way to answer questions. It also taught me that changing one line can fix the problem, create three new ones, or somehow do both.

Selected Projects

Four projects, shown in chronological order.

Each project gave me one new idea to learn and something interactive to build with it.

Project 01

Adventure Game

This was one of my first larger programming projects. I had to track inventory, location, and progression without losing any of them along the way. It showed me how several small pieces of code have to work together for the game to make sense.

Game StateInventoryNavigationProgression

Project 02

Blackjack Program

This project taught me how to randomize card draws, check conditions, and decide whether the player or dealer won. Blackjack was familiar enough that I could tell immediately when the code was doing something impossible.

RandomizationGame LogicConditionsDebugging

Project 03

Restaurant Ordering Game

I wanted to add visuals using Python’s Turtle graphics, so I made an incremental pizza-making game. It helped me combine graphics, user input, and a system that changed as the player kept going.

Python TurtleGraphicsUser InputIncremental Systems

Project 04

Multiplayer Board Game

For this project, I combined what I had learned about visuals and game logic into a four-player, turn-based board game. Keeping track of four players was exactly as simple as it sounds.
