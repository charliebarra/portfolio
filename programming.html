Programming Projects

Learning by building from scratch.

These projects show another side of my interest in systems. Programming helped me think about structure, logic, debugging, interfaces, and how one change can affect the rest of a program.

Selected Projects

Four projects, shown in chronological order.

Each project gave me a chance to learn a new coding idea and use it in something interactive.

Project 01

Adventure Game

One of my first larger programming projects, this game taught me how to track inventory, location, and progression. It helped me see how several small systems work together to create one experience.

Game StateInventoryNavigationProgression

Project 02

Blackjack Program

This project introduced me to randomized card draws, conditional logic, and win/loss rules. It was a useful way to see how code can model the rules of a familiar game.

RandomizationGame LogicConditionsDebugging

Project 03

Restaurant Ordering Game

In this project I learned how to implement visuals by using turtle graphics for python. So I made an incremental pizza-making game. It helped me combine graphics, user input, and a system that changes over time.

Python TurtleGraphicsUser InputIncremental Systems

Project 04

Multiplayer Board Game

Putting together all of my previously learned skills about visuals and coding logic, I made this multiplayer game. This four-player, turn-based board game combined visuals, turn management, game state, and player interaction.
