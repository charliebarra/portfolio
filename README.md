# Charlie Barra Portfolio Site

Static GitHub Pages-ready portfolio draft.

## How to preview
Open `index.html` in a browser.

## How to publish on GitHub Pages
1. Create a new public GitHub repository named something like `charlie-portfolio`.
2. Upload everything in this folder to the repository.
3. Go to Settings → Pages.
4. Under “Build and deployment,” choose “Deploy from a branch.”
5. Select `main` and `/root`, then Save.
6. GitHub will provide a public URL once the site deploys.

## Video notes
The uploaded videos are very large. For a live admissions portfolio, I recommend uploading final edited clips to YouTube as unlisted videos or compressing them before embedding. Then replace the placeholder text/links in `index.html` with the final video embeds.
